# Report Format

Default language is Chinese. Titles, category labels, event summaries, and follow-up suggestions should be Chinese. Keep English handles, links, product names, and short source snippets when useful for verification.

## Structure

```markdown
# Bagua AI Radar - YYYY-MM-DD HH:MM

## Run Notes

- Time windows: last 24h fresh, last 7d still heating.
- Platforms attempted: ...
- Warnings: ...

## Main Board

### 1. [label] Title

- Heat: 82/100 | Credibility: 71/100 | Category: 评测争议
- Why included: short scoring explanation.
- What happened: concise Chinese summary.
- 原文线索: short source snippet, usually preserved in the source language.
- Evidence chain:
  - X: handle, source type, URL
  - Reddit: subreddit/thread, URL
- Follow-up: what to watch next.

## Early Signals

Same shape, but use more cautious wording.

## Suggested Source Leads

List suggested Telegram channels or accounts discovered during collection, if any.
```

## Voice

Use a half-entertainment, half-analysis tone. Titles can be readable and slightly web-native, but evidence labels must be strict.

Preferred labels:

- `confirmed`: strong first-party or official source.
- `likely`: multiple public sources or strong evidence chain.
- `rumor`: socially hot but not verified.
- `weak signal`: thin evidence; included only because heat or source position is notable.
